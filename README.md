## Descripción del proyecto
SpeakUp es una aplicación móvil desarrollada en Flutter que esta enfocada en el aprendizaje de idiomas de forma didáctica e interactiva. La aplicación permite a niños y jóvenes aprender un vocabulario básico en diferentes idiomas mediante retos, preguntas y actividades dinámicas.
Nos permite aprender los idiomas inglés, portugués e italiano; cuenta con un sistema de preguntas y respuestas de opción multiple (a,b,c), cuenta también con vidad y puntajes el cual se evalua conforme la trivia avanza. Se buscó que la interfaz fuera interactiva y atractiva visualmente, además que para lograr esto se añadieron animacion y elementos visuales.

Se usaron las tecnologías de Flutter, Dart y diferentes providers para gestionar los datos de la aplicación completa.

## Instalación
1. Clonar el repositorio:
```bash
git clone https://github.com/AlondraYamiletCobosVidal/SpeakUp.git
```

2. Entrar al proyecto:
```bash
cd SpeakUp
```

3. Instalar dependencias:
```bash
flutter pub get
```

4. Ejecutar la aplicación:
```bash
flutter run
```
## ¿Cómo se usa?
La aplicación SpeakUp está diseñada para que los usuarios puedan aprender idiomas de una forma dinámica, sencilla e interactiva. El usuario instala el apk, para posteriormente inicializar la app desde si celular en dispositivo android 13 o superior.

Una vez iniciada la app, tendra que realizar un log in, se le enviara a una pantalla para elegir un idioma que desee aprender y cuatro categorías donde el usuario escogera la que desee(frutas, animales, colores y números).La trivia responderá con las vidas y el tiempo dado. 

En caso de fallar por tiempo o errores (haciendolo quedar sin vidas), podra volver a iniciar la trivia o volver al menu.

Así el usuario podra volver a escoger una y otra vez las demas categorias e idiomas, si el usuario desea salir de la aplicacion en el menu lateral izquierdo en el ultimo boton, podra cerrar la aplicación.
