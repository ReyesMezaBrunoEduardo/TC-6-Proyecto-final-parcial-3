## Descripción del proyecto
SpeakUp es una aplicación móvil desarrollada en Flutter que esta enfocada en el aprendizaje de idiomas de forma didáctica e interactiva. La aplicación permite a niños y jóvenes aprender un vocabulario básico en diferentes idiomas mediante retos, preguntas y actividades dinámicas.
Nos permite aprender los idiomas inglés, portugués e italiano; cuenta con un sistema de preguntas y respuestas de opción multiple (a,b,c), cuenta también con vidas y puntajes el cual se evalua conforme la trivia avanza. Se buscó que la interfaz fuera interactiva y atractiva visualmente, además que para lograr esto se añadieron animaciones e ilustraciones que llaman la atención de forma visual.

Se usaron las tecnologías de Flutter, Dart y diferentes providers para gestionar los datos de la aplicación completa.

## Instalación
1. Ingresar al enlace del repositorio en GitHub.

2. Descargar el proyecto presionando el botón verde llamado “Code” y posteriormente seleccionar la opción “Download ZIP”.

3. Una vez descargado el archivo ZIP, tenemos que descomprimir la carpeta en cualquier ubicación de la computadora.

4. Abrimos Visual Studio Code para poder tener acceso al código.

5. Seleccionar la opción “Abrir carpeta” y abrir la carpeta del proyecto recién descargado y descomprimido.

6. Verificar que Flutter y Dart estén instalados de forma correcta en el equipo, el cual es una computadora (es necesaria para el proceso de nuestra instalación).

7. Abrir la terminal con 'Ctrl + ñ' dentro del proyecto y ejecutar el siguiente comando para instalar todas las dependencias necesarias de la aplicación y así no nos marque algún tipo de error:
"flutter pub get"

8. Revisar el archivo de "pubspec.yaml" para verificar las versiones de Flutter y las dependencias utilizadas en el proyecto. Se recomienda utilizar una versión de Flutter compatible con la computadora o laptop para evitar errores durante la compilación.

9. Conectar un dispositivo celular a la computadora. El dispositivo debe ser Android 13 o alguna versión mayor.

10. Ejecutar la aplicación utilizando el siguiente comando en la terminal: "flutter run"

11. Esperar a que la aplicación compile e inicie automáticamente.

## ¿Cómo se usa?
La aplicación SpeakUp está diseñada para que los usuarios puedan aprender idiomas de una forma dinámica, sencilla e interactiva. El usuario instala el apk, para posteriormente inicializar la app desde su celular en dispositivo android 13 o superior.

Una vez iniciada la app, tendra que realizar un login, se le enviara a una pantalla para elegir un idioma que desee aprender y cuatro categorías donde el usuario escogera la que desee(frutas, animales, colores y números).La trivia responderá con las vidas y el tiempo dado. 

En caso de fallar por tiempo o errores (haciendolo quedar sin vidas), podra volver a iniciar la trivia o volver al menu. sí el usuario podra volver a escoger una y otra vez las demas categorias e idiomas, si el usuario desea salir de la aplicacion en el menu lateral izquierdo en el ultimo boton, podra cerrar la aplicación.

La aplicación de SpeakUp busca fomentar el aprendizaje básico de idiomas de manera accesible y divertida para usuarios de entre 7 y 13 años.
